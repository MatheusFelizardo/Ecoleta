O projeto em questão visa a criação de um site para localização/cadastro de pontos de coleta de resíduos recicláveis.

## Tecnologias utilizadas
### Front-End
* HTML
* CSS
* Javascript

### Back-End
* NodeJs
* SQLite

#### Também foi utilizado as dependencias abaixo:
- Express
- Nodemon
- Nunjucks
- Sqlite3
